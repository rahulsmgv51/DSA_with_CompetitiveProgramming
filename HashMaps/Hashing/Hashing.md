## How Hashing Works

Hashing is the process of assigning a unique value to an object using an algorithm to enable faster data access and retrieval.

### Key Points

- **hashCode() Method**: Implement the `hashCode()` method properly to optimize HashMap performance.
- **Return Value**: The `hashCode()` method returns an integer (4-byte value) generated by the hashing algorithm.
- **Purpose**: This hash value enables quick lookup and storage of objects in hash-based collections.

### Hashing of Strings

```java
int hash = 7;
for (int i = 0; i < strlen; i++) {
    hash = hash * 31 + charAt(i);
}
```
